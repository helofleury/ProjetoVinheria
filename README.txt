									Projeto Vinharia

	Projeto de criação de um site para uma vinharia. Nesse projeto criamos um site com páginas de vendas e informações sobre vinhos, informações de contato e dicas de harmonização.
	Na página de vendas, existem as categorias vinho branco, vinho rosé, espumantes e vinho tinto. Na harmonização, existem informações que dão dicas sobre o tipo de harmonização para fazer com o vinho, tanto com pratos doces como com pratos salgados.
